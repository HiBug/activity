### 接口文档

#### 约定

```java
1、所有接口必穿openId
2、斜体代表在上层最近结构体中
```

请求体结构

```json
{
  "function": "register",
  "openId": "safsdf",
  "param": {
    "mobile": "13122005258"
  }
}
```

返回体结构

```json
{
  "code": "0000",
  "message": "success",
  "data": {
    "activeFlag": true
  }
}
```
